### 3-2. 구현 기능

    - 카드
      - fetch 적용해 데이터를 받아 카드컴포넌트 구현
      - 카테고리 클릭 시 해당 카드 나오게 query이용하여 구현 
      - 카드 클릭 시 navigate 적용하여 detail페이지로 이동
      - 반응형 적용
